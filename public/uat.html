1. การเตรียมการ (Prerequisites)

1.1 เตรียม QR Codes สำหรับผู้เข้าร่วมงาน

  • ผู้รับผิดชอบ: (ระบุชื่อ)
  • ไปที่ URL: https://hoya-event.azurewebsites.net/prepare/
  • ทำการ Capture ภาพหน้าจอ QR Code ที่ไม่ซ้ำกัน
  • สิ่งที่ต้องได้: รูปภาพ QR Code จำนวน 20 รูป

1.2 เตรียมอุปกรณ์และบัญชีสำหรับ Staff

  • ผู้รับผิดชอบ: ผู้ทดสอบ 4 คน
  • เตรียมโทรศัพท์มือถือ จำนวน 4 เครื่อง
  • ใช้ Username ของ Staff ที่แตกต่างกัน 4 บัญชี
  • Login เข้าสู่ระบบ Staff App: https://hoya-evtstaff.vercel.app/

2. สถานการณ์การทดสอบ (Test Scenarios)

TC-01: การลงทะเบียนผู้เข้าร่วมงาน (Guest Registration)

ขั้นตอน รายละเอียด (Steps) ผลที่คาดหวัง (Expected Result) ผลการทดสอบ
1. Staff ทั้ง 4 คน เปิดหน้า "Register" ที่:
https://hoya-evtstaff.vercel.app/en/register		 หน้าจอแสดงกล้องพร้อมสแกน
2. Staff แต่ละคน ทำการสแกน QR Code คนละ 5 รูป จนครบ 20 รูป เมื่อสแกนสำเร็จ ระบบแสดงข้อความยืนยัน และ QR Code ที่สแกนแล้วไม่สามารถสแกนซ้ำได้

TC-02: การตรวจสอบข้อมูลบน Dashboard

ขั้นตอนรายละเอียด (Steps)ผลที่คาดหวัง (Expected Result)ผลการทดสอบ
1. เปิด Dashboard ที่:
http://hoya-eventify.vercel.app/en/public/dashboard		 Dashboard แสดงข้อมูลเริ่มต้น (Registered = 0)
2. สังเกต Dashboard ระหว่างที่ Staff กำลังสแกน และหลังจากสแกนครบ 20 รูป ตัวเลข Registered เพิ่มขึ้นแบบ Real-time และยอดรวมสุดท้ายต้องเท่ากับ 20

TC-03: การทดสอบ Import "Night Shift"

ขั้นตอนรายละเอียด (Steps)ผลที่คาดหวัง (Expected Result)ผลการทดสอบ
1. เข้าสู่ระบบ Admin ที่:
https://hoya-eventify.vercel.app/en		 Login เข้าสู่ระบบได้สำเร็จ
2. เข้าที่เมนูทางซ้ายชื่อ Registrations เข้าสู่หน้า Registrations ได้สำเร็จ และพบตัวเลือกสำหรับ Import
3. ทำการ Import ข้อมูล Night Shift
(Format: Excel ที่มี column "EmpID", "Clock in")		 ระบบแจ้งว่า Import สำเร็จ
4. ตรวจสอบข้อมูลที่ import ได้ที่:
https://hoya-eventify.vercel.app/en/registrations
โดยทำการ filter "night shift"		 พบข้อมูลที่ import เข้ามาแสดงในตารางอย่างถูกต้อง

TC-04: การออกรางวัล (Reward Drawing)

ขั้นตอนรายละเอียด (Steps)ผลที่คาดหวัง (Expected Result)ผลการทดสอบ
1. Dev/Admin ทำการล้างข้อมูลการออกรางวัลเดิม (ถ้ามี) ข้อมูลการออกรางวัลถูกล้างทั้งหมด
2. เปิดหน้าจอแสดงผลผู้โชคดี:
https://hoya-eventify.vercel.app/en/public/winners?session=day		 หน้าจอแสดงผลพร้อมสำหรับแสดงรายชื่อผู้โชคดี
3. Dev/Admin ทำการรันคำสั่งเพื่อออกรางวัลทั้งหมดจากเครื่อง local คำสั่งรันสำเร็จ และหน้าจอแสดงผลผู้โชคดี (ข้อ 2) แสดงรายชื่อผู้ได้รับรางวัลครบทุกรางวัล
4. ตรวจสอบผลการออกรางวัลในระบบ Admin ที่:
https://hoya-eventify.vercel.app/en/rewards		 มีรายชื่อผู้ได้รับรางวัลแสดงในตารางครบถ้วน และสถานะเป็น "Pending"

TC-05: การรับรางวัล (Prize Redemption)

ขั้นตอนรายละเอียด (Steps)ผลที่คาดหวัง (Expected Result)ผลการทดสอบ
1. Login ด้วยสิทธิ์ Super Admin ที่:
https://hoya-evtstaff.vercel.app/		 Login สำเร็จและเห็นเมนู "Prize Redemption"
2. เปิดหน้า Prize Redemption, ใส่รหัสพนักงานที่ได้รับรางวัล, ถ่ายรูป, และกด Submit ระบบบันทึกข้อมูลสำเร็จ และแสดงข้อความยืนยัน
3. ตรวจสอบข้อมูลและรูปภาพการรับรางวัลที่:
https://hoya-eventify.vercel.app/en/rewards		 สถานะของรางวัลเปลี่ยนเป็น "Redeemed" และมีรูปภาพที่ถ่ายแสดงขึ้นมา
4. ทดสอบใส่รหัสพนักงานที่ ไม่ได้รับรางวัล ในหน้า Prize Redemption ระบบแสดงข้อความแจ้งว่า "ไม่พบข้อมูลพนักงาน" หรือ "พนักงานไม่มีสิทธิ์รับรางวัล"
5. ทดสอบรับรางวัลซ้ำสำหรับพนักงานที่รับไปแล้ว ระบบแสดงข้อความแจ้งว่า "ของรางวัลถูกรับไปแล้ว"
